Design a modern, user-friendly web portal for patients of Manalo Medical Clinic, focusing on appointment booking, medical record viewing, notifications, and AI assistant interaction. The design must feel trustworthy, professional, and engaging while being simple to navigate. Avoid generic templates. Include subtle animations and interactive feedback for a smooth UX.

Brand & Theme:

Primary Colors: Blue (#0A2463, #1B4FD8, #3A86FF), White (#FFFFFF), Light Blue (#E8F1FF)

Typography:

Headings: Playfair Display — authoritative and elegant

Body: DM Sans — clean, readable, modern

Buttons/Badges: Uppercase small-caps, DM Sans

UI Style:

Glassmorphism cards on hero sections

Hover micro-interactions for buttons and cards

Smooth animations on loading and transitions (fade-ins, sliding panels)

Skeleton loaders for dynamic content

Accessibility compliant (WCAG 2.1 AA)

Web Pages & Modules:

Landing / Home Page

Hero section introducing clinic: “Welcome to Manalo Medical Clinic — Your Health, Our Priority”

Quick CTA buttons: “Book Appointment” | “Login / Register”

Brief services overview: Routine Check-ups, Specialist Consultations, Lab & Diagnostic Interpretation

Animated card highlights for each service

Interactive map preview with “Get Directions”

Patient Sign-up / Registration

Form fields: Full Name, Email, Contact Number, Address, Gender, Date of Birth, Password

Animated input validation and success states

Profile completion progress bar after registration

Patient Login

Email + password login

“Remember Me” toggle

Password reset link with animated confirmation messages

Patient Dashboard

Upcoming Appointment Banner: Animated card showing next appointment details (doctor, date, time, status)

Quick Book Appointment Button: Floating action button with hover animation

Appointment History Table: Sortable, filterable, paginated with status color indicators (Pending = amber, Approved = teal, Rejected = red, Completed = blue, Cancelled = gray)

Notifications Feed: Slide-in panel with timestamped updates, mark-all-as-read animation

AI Chat Assistant: Floating chat bubble with animated entry; AI responds in branded chat bubbles

Appointment Booking Module

Multi-step animated form:

Personal Info (pre-filled)

Select Date & Time (calendar with available slots highlighted)

Consultation Type & Reason

Review & Confirm (summary card with hover animations)

Real-time slot availability feedback

Patient Records / Consultation History

Card-based layout showing past consultations

Animated expand/collapse for details (diagnosis, prescription, notes)

Notifications / Reminders

In-app notification panel with smooth slide-in/out animations

Color-coded by type: Approved = teal, Rejected = red, Reminder = blue

Interactive Clinic Map

Embedded Google Map preview

Custom navy blue pin for clinic location

Animated info card overlay: clinic name, address, phone, schedule, buttons “Get Directions” / “Copy Address”

UX / UI Requirements:

Desktop-first, fully responsive on tablet and mobile

Smooth hover effects and transitions for buttons, cards, and tables

Feedback animations for user actions (form submit, login success/failure, booking confirmation)

Avoid showing any admin/secretary functionality

Maintain professional, modern look while being approachable

Deliverables:

Figma-ready design files for all pages and modules

Interactive prototype showing hover effects, animations, and transitions

Include all screens: Home, Registration, Login, Dashboard, Appointment Booking, Records, Notifications, Map, AI Chat

Optional:

Subtle animated page load (fade-in sections)

Skeleton loaders for dynamic data (appointments, history, notifications)
